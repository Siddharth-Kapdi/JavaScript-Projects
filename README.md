# JavaScript Practice Projects

A curated collection of small JavaScript apps created for learning and practicing frontend
development. Each project includes HTML, CSS, and JavaScript files to demonstrate interactivity, API
usage, and basic state management.

## Projects Overview

### 1. Expense Tracker

- Track income and expenses with live balance updates.
- Stores data in `localStorage` so transactions persist between page reloads.
- Includes add and delete transaction functionality.

### 2. GitHub Account Finder

- Search for GitHub users by username.
- Displays profile details such as avatar, bio, followers, repositories, and social links.
- Fetches repository metadata from the GitHub API.

### 3. Quiz App

- A simple client-side quiz with predefined questions.
- Shows score, progress, and feedback for each round.
- Includes question navigation and restart support.

### 4. Quiz App API

- Fetches quiz questions from the Open Trivia Database (OpenTDB) API.
- Randomizes answer options and tracks score.
- Shows a results screen with customized feedback.

## How to Run

1. Open the project folder in your browser.
2. Open the folder for the app you want to try.
3. Open `index.html` in a browser.

## Notes

- These apps are designed for learning JavaScript concepts such as DOM manipulation, event handling,
  and working with APIs.
- The folder names match the project names in this README.
